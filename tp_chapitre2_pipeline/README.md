# TP Chapitre 2 : Pipelines agiles, versionnement et orchestration
Ce projet contient un pipeline local d'ingestion, de validation et de transformation de données.

## Technologies utilisées
- **dlt** pour l'ingestion de données
- **dbt** pour la transformation au sein de DuckDB
- **Dagster** pour l'orchestration globale
- **Git & GitHub Actions** pour le versionnement et la CI
